# Linear Tasks for Nahuel Soria Portfolio Project

## HIGH PRIORITY TASKS

### 1. Calendly Account Setup
**Title:** Configure Calendly integration for portfolio contact methods
**Labels:** `manual-task` `high-priority` `portfolio-2026`
**Description:** Set up Calendly account with 3 meeting types for portfolio contact
**Acceptance Criteria:**
- [ ] Create Calendly account
- [ ] Configure "Idea de App/SaaS" meeting type (15 minutes)
- [ ] Configure "Auditoría Técnica" meeting type (30 minutes) 
- [ ] Configure "CTO as a Service" meeting type (45 minutes)
- [ ] Set availability (morning or afternoon hours)
- [ ] Update components/calendly-button.tsx with actual Calendly URLs
- [ ] Test meeting booking flow end-to-end
**Estimated Time:** 2-3 hours

---

### 2. Google Analytics 4 Configuration
**Title:** Set up GA4 tracking for nahuelsoria.com
**Labels:** `manual-task` `high-priority` `portfolio-2026`
**Description:** Configure Google Analytics 4 for production tracking
**Acceptance Criteria:**
- [ ] Create GA4 property for nahuelsoria.com
- [ ] Get NEXT_PUBLIC_GA_ID from GA4 setup
- [ ] Add GA4 ID to .env.local
- [ ] Add GA4 ID to Vercel Environment Variables
- [ ] Deploy to production and verify tracking is working
- [ ] Test real-time events in GA4 dashboard
**Estimated Time:** 1-2 hours

---

### 3. Project Screenshots Collection
**Title:** Collect real project screenshots for case studies
**Labels:** `manual-task` `high-priority` `portfolio-2026`
**Description:** Obtain and organize real screenshots for all portfolio projects
**Acceptance Criteria:**
- [ ] Get What I Wish app screens (mobile/desktop views)
- [ ] Get Numerai dashboard screens (analytics interface)
- [ ] Get Picaday photo journal interface screenshots
- [ ] Get CondorFX trading platform screenshots
- [ ] Get Bruselascambio exchange interface screenshots
- [ ] Organize images in /public/projects/ folder with proper naming
- [ ] Update CaseStudies.tsx to use real project images
- [ ] Ensure all images are optimized for web (WebP format, proper sizing)
**Estimated Time:** 4-6 hours
